Detailed Description

template<typename V>
struct ExternalStringHashMap< V >

A template for hash_maps keyed by a const char* that is maintained within the values.

Use like: ExternalStringHash<Foo*>::Type foos; to make foos into a hash mapping const char* => Foo*.
